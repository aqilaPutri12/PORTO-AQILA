Tentang Saya

Nama saya Aqila Putri. Saya lahir di Bogor pada tanggal 12 Juli 2006. Saya siswi SMK Wikrama Bogor, dengan jurusan Rekayasa Perangkat Lunak dan Gim (PPLG). PROJECT

TABUNGAN

Website tabungan merupakan website yang dibuat untuk mendata pemasukan dan pengeluaran pengguna. Website akan otomatis menghitung jika terdapat data pemasukan atau data pengeluaran baru. Website tabungan dibuat menggunakan bahasa pemograman laravel.

PEMINJAMAN LAPTOP

Peminjaman laptop merupakan website yang dibuat khusus untuk siswa SMK Wikrama Bogor yang meminjam laptop di sekolah. Data siswa yang meminjam laptop akan di masukan oleh laboran, dan yang dapat membuat akun laboran adalah admin. Website peminjaman laptop dibuat menggunakan bahasa pemograman laravel.

PPDB SMK WIKRAMA

PPDB SMK Wikrama merupakan website Penerimaan Peserta Didik Baru (PPDB), cara kerja website PPDB SMK Wikrama yaitu calon siswa dapat mendaftarkan diri secara online, dengan cara register di web tersebut, kemudian siswa akan mendapatkan username dan password untuk login ke akun siswa. Jika sudah login, maka siswa akan melihat informasi-informasi terbaru mengenai administrasi, pengumuman penerimaan, wawancara, dll. Website PPDB SMK Wikrama dibuat menggunakan bahasa pemograman laravel.

iKANTIN

Ikantin merupakan website kantin sekolah, cara kerja website ikantin yaitu siswa dapat melihat daftar menu sekaligus harga menu tersebut, jika siswa membeli beberapa menu di kantin, website dapat menghitung total menu yang di pesan oleh siswa tersebut. Website ikantin dibuat menggunakan bahasa pemograman PHP.
